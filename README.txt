Description:
This is Maven Project.
It consists of TestingAssignment class, which includes mathods to test features of https://the-internet.herokuapp.com website.

Requirements:
1. Maven
2. JDK 8
3. Chrome Browser

Instructions to run the tests:
1. Download the test-for-assignment Maven project.

2. In cmd navigate to the project directory and run in cmd:
mvn clean test

3. Generated TestNG reports: 'index.html' and 'emailable-report.html' can be found in below directory:
PROJECT_DIRECTORY\target\surefire-reports\

where:
PROJECT_DIRECTORY- directory of downloaded test-for-assignment Maven project
